# Password Generator 🔐

A simple Python command-line tool that generates **secure, random passwords** using combinations of uppercase letters, lowercase letters, digits, and special characters.

This project demonstrates basic Python concepts such as **string manipulation, randomization, user input, and modular code structure** while also highlighting the importance of strong password practices.

---

## Features

- Generate random secure passwords
- Customizable password length
- Includes:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Lightweight and easy to run
- No external dependencies
- Suitable for beginners learning Python

---

## Why Use a Password Generator?

Weak passwords are one of the most common causes of security breaches.  
Strong passwords should:

- Be **long**
- Include **multiple character types**
- Be **random and unpredictable**

This generator helps create passwords that are much harder to guess or brute-force.
